üîê Security Information

üîë Environment Variables

The following environment variables should be set in your Render deployment for security:

  • ADMIN_USERNAME: Your admin username
  • ADMIN_PASSWORD_HASH: SHA-256 hash of your admin password
  • ADMIN_SECRET_KEY: Secret key for session management

üîí Password Security

To generate a password hash, you can use Python: 

import hashlib
password = "your_password_here"
hashed = hashlib.sha256(password.encode()).hexdigest()
print(hashed)
